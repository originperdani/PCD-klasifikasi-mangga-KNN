# TO-DO: Perbaikan Tampilan Website Klasifikasi Kematangan Mangga

## Langkah-langkah:

1. Perbarui file `static/css/style.css`:

   - Ganti skema warna menjadi lebih modern dan menarik.
   - Tingkatkan tipografi dengan font yang lebih elegan dan pengaturan ukuran serta jarak.
   - Tambahkan desain responsif agar tampilan maksimal di berbagai ukuran layar.
   - Rapikan jarak antar elemen dan tata letak agar lebih rapi dan enak dipandang.
   - Tambahkan animasi halus pada tombol dan gambar untuk interaktivitas yang lebih baik.

2. Perbaiki struktur HTML di `templates/index.html` (jika diperlukan):

   - Tambah atau sesuaikan wrapper atau kelas CSS agar styling lebih optimal dan konsisten.
   - Pastikan setiap bagian hasil (gambar, fitur, klasifikasi) terorganisir dengan baik.

3. Tes hasil perubahan dengan membuka website dan memastikan tampilan sudah diperbagus secara signifikan.

4. Lakukan perbaikan tambahan jika ada feedback atau masalah ditemukan.

Catatan:

- Fokus utama pada peningkatan tampilan dan user experience tanpa mengubah fungsi utama.
